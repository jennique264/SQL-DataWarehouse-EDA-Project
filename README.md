# Sales Data Exploratory Analysis Using SQL

## Overview
This project focuses on performing Exploratory Data Analysis(EDA) on retail sales data using SQL Server. 
The project explores and summarizes data across customers, orders, products, and sales transactions to apply data-driven decision-making.


## Objectives

* Analyze sales performance and business KPIs
* Understand customer demographics and purchasing behavior
* Evaluate product and category performance
* Identify top-performing and low-performing products
* Practice SQL-based data exploration and analytics

## Dataset

The dataset consists of:

* Customer information
* Product information
* Sales transactions
* Order records

The data is organized into fact and dimension tables to support analytical queries.
